# Student Success Platform

## Folder Structure
- `infra/` → Docker, env configs
- `db/` → SQL scripts, migrations
- `hasura/` → Hasura metadata
- `web/` → Frontend
- `automations/` → n8n workflows
- `docs/` → Docs, PPTs, reports

## Roles
- **DB** → db/
- **Auth** → infra/
- **Hasura/API** → hasura/
- **Frontend** → web/
- **Comms/Automation** → automations/
- **QA + Docs** → docs/
